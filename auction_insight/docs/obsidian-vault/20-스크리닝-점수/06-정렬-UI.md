# 검색 정렬 · 앱 표시

← [[00-스크리닝-점수-MOC|목차]]

## API 정렬 (`search_lots`)

기본: `sort=score` (앱 “추천순”)

| sort | 우선순위 |
|------|----------|
| **score** (기본) | ① 좌표 있는 물건 ② `total_score` 내림차순 ③ 마감 빠른 순 |
| **deadline** | ① 마감 빠른 순 ② 좌표 ③ `total_score` |
| **discount** | ① `discount_vs_appraisal` 큰 순 ② 좌표 ③ 마감 |

코드: `backend/app/services/lots.py`

## 앱에서 보이는 곳

| 화면 | 표시 |
|------|------|
| 홈 정렬 칩 | 추천순 / 마감임박 / 할인율 |
| 목록 타일 | 우측 **점수** = `total_score`; 감정할인 ≥10%면 칩 |
| 물건 상세 | 감정·시세 대비, 종합·상권·마감임박 |
| 분석 랩 시세 탭 | 연결된 lot의 할인·`total_score` 스냅샷 |

## 점수에 안 들어가는 것

- `fail_count` (유찰) — 필터·“유찰 N회” 칩만
- 권리·유치권 플래그 — 위험칩·가이드 문구 (점수식 아님)
