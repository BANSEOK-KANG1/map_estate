# 임계값 · 치트시트

← [[00-스크리닝-점수-MOC|목차]]

점수 **공식**과 별도로, UI·팁에서 쓰는 컷오프입니다.

## 할인 · 마감 칩

| 조건 | 어디에 |
|------|--------|
| 감정/시세 할인 ≥ **15%** | 목록·상세 하이라이트 태그 |
| 감정 할인 ≥ **10%** | 목록 타일 칩 |
| 감정 할인 ≥ **30%** | 초보 전략 팁 (“기회일 수도 함정일 수도”) |
| D-day ≤ **7** | `D-n` / 마감 칩 |
| 시세 표본 &lt; **8** | “시세표본 적음” 태그 |

## 시세 신뢰도 (표시용)

| 표본 수 | confidence |
|---------|------------|
| ≥ 20 | high |
| ≥ 8 | medium |
| &lt; 8 | low |

## 종합 가중치 (다시 보기)

```
시세할인 35% + 감정할인 25% + 인프라 25% + 긴급도 15%
```
