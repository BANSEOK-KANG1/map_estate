# 마감 긴급도

← [[00-스크리닝-점수-MOC|목차]]

입찰 마감(`bid_end_at`)이 가까울수록 점수가 높습니다. (0~100)

## 구간표

| 남은 시간 | `urgency_score` |
|-----------|-----------------|
| `bid_end_at` 없음 | **0** |
| 이미 마감 (≤0초) | **100** |
| 3일 미만 | **95** |
| 3일 ~ 7일 미만 | **85** |
| 7일 ~ 14일 미만 | **65** |
| 14일 ~ 30일 미만 | **40** |
| 30일 이상 | **10** |
